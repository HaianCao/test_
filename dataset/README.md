# Thư mục dữ liệu (Dataset)

Thí sinh cần tải các tập dữ liệu từ nền tảng cuộc thi trên Kaggle và đặt vào thư mục này để có thể chạy được starter kit và script đánh giá.

Cấu trúc các file yêu cầu bên trong thư mục này:

- `X_train.csv`: Tập dữ liệu đầu vào cho quá trình huấn luyện.
- `Y_train.csv`: Tập dữ liệu mục tiêu (nhãn) tương ứng cho quá trình huấn luyện.
- `X_val.csv`: Tập dữ liệu đầu vào cho quá trình kiểm định.
- `Y_val.csv`: Tập dữ liệu mục tiêu (nhãn) tương ứng cho quá trình kiểm định.
- `X_test.csv`: Tập dữ liệu đầu vào cho quá trình kiểm thử (dùng để tạo file nộp).
